2017-12-16 JS study Scope

JS Study - Scope, Closer

  1. 지역변수 => 블록수준, 함수수준의 언어냐를 물어본다.
  2. 함수수준은 함수레벨(범위)안에서 이루어졌다는의미.
  3. { 중괄호로 묶인것을 블록수준이라고한다.
지역변수는 함수내에서 전역변수보다 높은 우선순위를 지닌다.
전역변수는 함수의 외부에서 선언된 모든 변수의 전역 범위이다. 하지만 가급적 전역변수의 사용을 자제해야한다. 이를 방지? 하기 위해서 JS ES6에서는 let과 const를 도입했다.
만약 변수가 최초 선언 없이 초기화가 되었다면, 이 변수는 자동으로 전역 컨텍스트에 추가가된다.
함수안에서 age를 var age로 선언을 하면 지역변수가 된다. 하지만 없으면 JS는 이를 전역변수로 이해를 한다.
