자바스크립트 hello world 출력

실행하는 법

  • 크롬에서 개발자 도구를 열고 콘솔에서 실행
  • node를 실행한 후 REPL에서 실행
  • node filename.js 으로 실행
  • html과 js 생성 후 브라우저로 드래그 앤 드롭

리눅스 명령어

ls: 디렉토리의 내용 보기
pwd: 현재 디렉토리의 위치확인
cd: 디렉토리 이동
mkdir: 디렉토리 생성
cp, mp, rm: 파일, 복사, 이동, 삭제
