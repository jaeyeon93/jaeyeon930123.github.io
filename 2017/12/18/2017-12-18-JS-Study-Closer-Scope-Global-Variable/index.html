JS Study - Scope

Scope란?

  • 함수의 유효범위이다.

전역변수

  • 함수 외부에서 선언된 변수이다. 많이 쓰면 나쁘다.
  • 최상위 객체에 속하는 변수이다.
  • global의 프로퍼티다.

지역변수

  • 함수 안에서 선언된 변수
  • 함수 외부에서는 쓸 수 없다.

자동 전역 변수

  • 함수 안에서 var없이 선언을 하면 자동 전역 변수가 된다. 하지만 절대 그러지 말자.

같은 이름이면 지역변수가 전역변수보다 우선순위가 높다.

함수의 매개변수
